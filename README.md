## What is Pulsar?

Pulsars are among the most precise objects in the universe — neutron stars that emit perfectly rhythmic radio bursts with clockwork regularity. They are powerful, predictable, and impossible to misread. Pulsar, the language, is built around the same idea.

Pulsar is a statically-typed, expression-oriented programming language that transpiles to JavaScript. Inspired by the readability of Python and the structural discipline of Swift, it is designed with one principle: that complexity should never come at the cost of clarity. Whether you are writing a simple function or modeling a layered data structure, the syntax stays close to the idea behind it — readable, intentional, and clean.

The compiler enforces type safety, scope rules, and return path correctness at compile time, so errors surface before your program ever runs. Pulsar is built for students and new programmers who want a language that grows with their thinking, not against it.
